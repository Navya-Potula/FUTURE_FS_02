# FUTURE_FS_02 - Mini CRM Project

## 📌 Project Title
Mini CRM System (Customer Relationship Management)

## 📖 Description
This project is a simple CRM system that helps a business manage leads coming from website contact forms. It allows admin users to track, update, and manage customer leads efficiently.

## 🚀 Features

- Lead listing (name, email, source, status)
- Update lead status (New → Contacted → Converted)
- Add follow-up notes for each lead
- Secure admin login system
- Dashboard to manage all leads
- Local storage based data persistence

## 🛠️ Tech Stack

- React (Frontend)
- JavaScript (ES6)
- HTML & CSS
- LocalStorage (for data persistence)
- Framer Motion (UI animations)

## 🔐 Authentication
Only logged-in users can access and manage leads.

## 📊 Core Functionality

- Add and view leads
- Update lead status
- Track customer interactions
- Store data securely in browser storage

## 💡 Bonus Features (if implemented)

- Search & filter leads
- Timestamp tracking
- Basic analytics (total leads, conversions)
